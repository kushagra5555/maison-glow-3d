# Maison Glow 3D

Premium 3D landing page mockup for a boutique beauty studio, makeup artist, and luxury beauty parlor.

## Highlights

- Interactive Three.js hero and 3D beauty orbit scene
- Luxury editorial typography and responsive visual system
- Filterable portfolio gallery
- Before and after reveal slider
- Package comparison and client journey sections
- Sticky mobile booking CTA and WhatsApp enquiry CTA
- GitHub Pages deployment workflow

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
